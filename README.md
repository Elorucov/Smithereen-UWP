# Smithereen UWP app

This is an UWP app for [Smithereen — a federated social network](https://github.com/grishka/Smithereen). Currently it is in the initial stages of development.

Minimal supported OS version is __Windows 10 Anniversary update (version 14393)__. It's can be installed on Windows 10 Mobile devices.

### Building
1. Download and install Visual Studio (between 2017 and 2026) with Universal Windows workloads and Windows 10 SDK (version 19041).
2. Clone this repository.
3. Open .sln file, build and run.
